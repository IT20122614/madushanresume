Project Description

H&O is a company that provides home and office services. Such as repairing electric items, cleaning, painting, interior designing, etc. This company is like a broker company. This company introduced different service providers to the clients. In return, they take payment from service providers to post their advertisements—the company using a manual file base system to keep its records. They wanted a web application to make their work easy. The requirements were gathered, and an automated system was decided with eight main functionalities. This new system was designed using the latest technologies. To develop the back-end of the system, Node.js was used. For the API, Express.js was used, and for the system's front-end, React.js, CSS was used. This system using MongoDB to store all data of the system. All system interfaces are user-friendly and user-attractive. It's easy to navigate through the system. The new system fulfills all their requirements, and now it's elementary for their customers. The new system will help the company do better and more efficiently than before, increasing their business.
