Project Description

“NITROTECH” is a mobile app for computer solutions shop. First user can sign up with the app and the user need to login to the app. They can add their details from address book in admin panel. Administrators can manage all items in the system such as adding items, updating items and delete items. When administrator logged into the system user can go to the admin panel from “admin panel” button. In admin panel admin can manage laptops and spare parts and admin can manage all type of orders. By clicking an order admin can see the order details. In order details admin can get the payment slip and he can download the slip using download button. In pc build order details he can the build quotation details as well. User can navigate through the system. In laptops user can check the laptops available in the app and they can add them to cart if they want to buy them. They can add spare part as the same way. After that they can go to the cart in the app. In cart they can remove items if they want to and after that they can go to the payment and they have to pay for the items they want to buy and they can upload the image to the app. then admin will get these orders. In build pc users can build a pc as the way they want. First, they have to create a quotation by giving a name for the quotation. Then they can go to the quotation items and add items to the quotation. They can see the total amount top right side of the screen. By clicking “Add items” button they can go to item select list. They can select items by its model, and they can decide how many items they want, and they can give the quantity and click add button then that item will be sent to the quotation list. In pc build interface user can click the “perches” button to pay for that build then they can pay for the build and upload the slip. Then admin will get that order.
